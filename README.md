# unicable-dvbs-hls-gateway
Self-contained SAT-to-HLS gateway for hotel and property TV systems — multi-tuner DVB-S2/Unicable ingest, per-channel hls making, and a lightweight web UI, built to run reliably as adapter count scales.

The HLS streamer was created as an experimental hobby project focusing on the reception of DVB-S2 satellite channels. 
and publishing them as standard HLS streams. Over time, it evolved into a self-sufficient streaming tool for hotels or IPTV

The project was created with simplicity in mind.

Instead of requiring powerful servers or complex container environments,
Streamer HLS is designed to run reliably even on modest hardware such as
small thin clients equipped with PCIe DVB adapters.

## Features

- DVB-S2 multi-adapter support
- JESS / Unicable II (EN50607)
- Per-channel HLS streaming
- Independent tuner services
- Lightweight Web Control Panel
- IP access control
- Channel scanning
- Low resource usage

---

## Why?

Because satellite streaming shouldn't require enterprise hardware.

Streamer HLS focuses on one task:

Receive DVB channels.

Publish HLS streams.

Stay simple.
